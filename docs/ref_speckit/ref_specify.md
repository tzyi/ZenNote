ZenNote 卡片筆記 APP 產品功能總覽

# 1. 主畫面（Main Feed Screen）
時間流卡片設計：以 FlashList 垂直排列，時間倒序顯示，支援無限滾動。
卡片架構：每張卡片含精確時間戳、內容預覽、多行顯示（字數超過自動折疊）、標籤徽章（#TODO、#GitHubCopilot）、更多操作（...）。
互動行為：
點擊卡片展開全內容並進入編輯模式。
左右滑動卡片：左刪除、右設為精選。
長按卡片：氣泡選單（刪除、分享、置頂）。
下滑載入更多卡片，Reanimated 淡入動畫。
懸浮新增按鈕（FAB）：右下角綠色漸層 + 按鈕，觸發文字編輯器（Bottom Sheet）。
頂部導航：漢堡選單（側邊欄）、快速搜尋圖示。
底部熱力圖與統計：
12x12 格子陣列熱力圖，顯示每日記錄頻次。
大字體統計（今日/本週/總數）。
搜尋按鈕。
驗收標準：
首次載入 <500ms 顯示 10 張卡片。
滾動 60fps，暗黑模式自動適配。
無網路時顯示本地資料。
技術建議：FlashList 虛擬化列表、NativeWind 樣式、Reanimated 卡片滑動、Zustand 狀態管理。


# 2. 文字編輯器（Text Editor - Bottom Sheet）
底層彈出片式設計：非全屏，讓使用者感知背景情境。
極簡輸入介面：預設顯示「現在的想法是...」佔位符。
工具列：
標籤插入（#）、圖片上傳、語音錄製（麥克風圖示）、格式化（粗體、無序列表）、進階功能選單。
心情滑桿：右側浮動，5 星綠色按鈕，滑動變色卡片邊框。
標籤建議：底部快速標籤（#工作 #想法）。
互動行為：
即時自動儲存草稿（每 2 秒 debounce）。
@ 觸發提及清單，# 自動補全標籤。
ESC 或返回：確認儲存並回主畫面。
驗收標準：
語音轉文字準確率 >90%。
離線自動儲存，同步時上傳。
鍵盤彈出不擋工具列，支援 Android 硬體鍵盤。
技術建議：react-native-markdown-editor、react-native-voice、KeyboardAvoidingView、AsyncStorage 草稿暫存、gorhom/bottom-sheet。


# 3. 搜尋頁面（Search Page）
多維度篩選：以 Chips 組件呈現屬性篩選（無標籤、有鏈接、有圖片、有語音、那年今日）。
即時搜尋：頂部搜尋框，輸入即觸發過濾邏輯，debounce 搜尋。
結果列表：同主畫面卡片樣式，結果計數顯示。
自訂篩選：日期 picker、標籤多選。
互動行為：
輸入 >3 字即顯示結果，空格分隔多關鍵字 AND 搜尋。
點擊篩選更新列表（動畫過渡）。
無結果顯示提示。
驗收標準：
搜尋延遲 <100ms（本地 SQLite 全文檢索）。
支援 #標籤 精確匹配、歷史搜尋記錄。
技術建議：SQLite + FTS5 全文搜尋、Zustand 搜尋狀態、Reanimated 過渡動畫、DateTimePicker。


# 4. 左邊側邊欄（Side Drawer）
個人看板屬性：整合使用者統計與導航管理。
數據可視化：
熱力圖（GitHub 格狀）、統計指標（筆記總數、標籤總數、堅持天數）。
功能導航：全部筆記、每日回顧、AI 洞察等進階功能入口。
標籤管理：列出所有標籤，拖曳排序，按筆記數量排序，點擊跳轉聚合視圖。
Pro 會員標記：頂部顯示用戶名與 Pro 升級入口。
設定管理：同步、隱私鎖、主題切換、密碼鎖、通知、匯出。
互動行為：
左滑手勢開啟，右滑或點遮罩關閉。
點擊標籤跳轉篩選結果，拖曳排序並儲存。
Android 返回鍵關閉。
驗收標準：
開啟動畫 <200ms，標籤同步即時。
技術建議：GestureHandler 側邊欄、NativeWind 樣式、Zustand 狀態管理。
